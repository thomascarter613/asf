# Anamnesis

## Purpose

Anamnesis is the Charon rehydration engine.

It determines what context should be loaded for a session, run, work packet, or agent role.

## Owns

Anamnesis owns:

1. rehydration rules;
2. startup context requirements;
3. task-specific context selection;
4. inclusion and exclusion logic;
5. context budget application.

## Status

Rehydration rules are documented.

No executable rehydration engine exists yet.
