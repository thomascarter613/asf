# Implementation Checklist

Use this checklist before calling implementation work complete.

```text
[ ] Work packet exists.
[ ] Scope is clear.
[ ] Non-goals are clear.
[ ] Required context was read.
[ ] Files created match the work packet.
[ ] Files modified match the work packet.
[ ] Files to avoid were avoided.
[ ] No unrelated scope was added.
[ ] No destructive action occurred without approval.
[ ] No dependency was added without review.
[ ] Verification was run or honestly recorded as not-run.
[ ] Documentation was updated if required.
[ ] Handoff need was assessed.
[ ] Memory candidate need was assessed.
[ ] Commit recommendation is present.
```
