# Work Protocols

## Purpose

This directory contains Foundry work protocols.

A protocol defines execution discipline for a category of work.

Protocols prevent short directives from becoming vague, unbounded, or unverifiable implementation.

## Files

```text

```

## Rule

A work packet defines scope.

A protocol defines discipline.
