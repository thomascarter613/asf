# Retrospectives

## Purpose

This directory will store retrospectives and reusable lessons from Foundry work.

Retrospectives may produce memory candidates.

They do not become canonical memory automatically.

## Possible Retrospective Topics

1. what worked;
2. what failed;
3. what was confusing;
4. what should be automated next;
5. what should become a memory candidate;
6. what should become a repo contract;
7. what should become a policy update.
