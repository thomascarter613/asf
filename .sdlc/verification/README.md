# Verification Records

## Purpose

This directory stores verification records.

Verification records capture evidence of completion, failure, partial completion, skipped checks, or blocked verification.

## Template

Use:

```text
.sdlc/verification/verification-record-template.md
```

## Rule

Verification is completion evidence.

Unverified work must not be presented as complete.
